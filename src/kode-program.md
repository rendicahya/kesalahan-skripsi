# Panduan Penulisan Kode Program

Bagian ini mengatur tata cara pencantuman dan format penulisan kode program (*source code*) di dalam laporan skripsi.

---

### 1. Urgensi Pencantuman Kode Program

Memasukkan kode program ke laporan skripsi hanya untuk mempertebal halaman laporan tanpa menambah makna atau esensi penjelasan, sama sekali tidak dibenarkan. Hanya kode program yang benar-benar penting, krusial, atau merepresentasikan algoritme utama saja yang perlu dimasukkan ke laporan skripsi.

---

### 2. Format Rataan Teks (*Alignment*)

Teks atau baris kode program di dalam laporan wajib ditulis dengan format **rata kiri (*align left*)**, bukan rata kanan-kiri (*justify*). Hal ini bertujuan untuk menjaga keterbacaan struktur, indentasi, dan sintaks dari kode tersebut.

---

### 3. Ketentuan Label Penamaan

Kode program yang dimasukkan ke dalam dokumen laporan tidak boleh diberi label sebagai Gambar ataupun Tabel. Silakan baca dan pelajari [Buku Panduan Skripsi Versi 3.0](https://file-filkom.ub.ac.id/fileupload/assets/upload/filemanager/Skripsi/PanduanSkripsiFilkom-v3.0.pdf) halaman 51 mengenai ketentuan baku format penyisipan kode program.
